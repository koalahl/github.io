如何在swift项目中创建Objective-C Bridging header

iOS开发
  • Objc Bridging Header
  • swift

在swift工程中，有时候我们也需要使用OBJC代码写的第三方库或者工具类等，这时就需要创建objc文件到swift的桥接header。
现在有两种方法创建Objective-C Bridging Header:

1.Create Objective-C File as Header


Go to File > New > File and Select Objective-C:


然后一直next，最后出现如下图时，点击yes：

这种方式会创建出.h和.m文件。

2. Create Objective-C Header File Directly


Go to File > New > File and Select Objective-C Header File:

最后，两种方法创建的.h文件中import你将使用的类。

一般在项目中我们使用第二种方法。
