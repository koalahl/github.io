在此总结一下常用排序算法的代码实现

#include <iostream>
using namespace std;
typedef int ElemType;

/*
1、插入排序
（1）直接插入排序算法
算法思想：将等排序列划分为有序与无序两部分，然后再依次将无序部分插入到已经有序的部分，最后

就可以形成有序序列。
操作步骤如下：
1）查找出元素L（i）在表中的插入位置K；
2）将表中的第K个元素之前的元素依次后移一个位置；
3）将L（i）复制到L（K）。
*/
时间复杂度为：O(n^2)
